Développement Full Stack : une application complète(zoo)  comportant :

Un serveur PHP :

Une API REST PHP

Une partie administration PHP

Un site front-end réalisé entièrement avec REACT (Axios pour récupérer les datas du serveur)

L'objectif de ce projet est debâtir un site web complet pour un Zoo, en partant de zéro,
et de mettre en place un code robuste et de qualité professionnelle.



Le projet s'articule autour de 4 parties :

Analyse du besoin

Création du serveur et de l'API REST PHP

Création de l'application Front-end REACT

Création de la partie administration en PHP


Le projet est axé sur la mise en relation de la partie front-end REACT avec une partie back-end PHP.

D'autres technologies utilisées: HTML, Bootstrap, MySQL, ...
